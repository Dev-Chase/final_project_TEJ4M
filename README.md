# TEJ4M - Projet Finale - Reconnaissance Faciale

## Configuration de l'Environnement
* Créer un Environnement Virtuel Python:
    * Éxecute **_python -m venv venv_** dans le dossier du code
* Entrer dans l'environnement: **_source venv/bin/activate_**
* Installer les bibliothèques nécessaires (_pip install_ ___)
    * opencv-python
    * face_recognition
        * si ça demande pour exécuter un commande git, éxecute-la, ensuite installer _setuptools_ avec **_pip install setuptools_**
    * gpiozero
    * imutils

## Exécution
* Exécuter __*python main.py*__
